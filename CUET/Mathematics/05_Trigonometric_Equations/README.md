## General Solution
- $sinx = sin\alpha \to x = n\pi + (-1)^n \alpha$ 
- $cosx = cos\alpha \to z = 2n\pi \pm  \alpha$
- $tanx = tan\alpha \to x = n\pi + \alpha$
- $x = n\pi + \alpha \to \begin{cases} sin^2x = sin^2\alpha \\ cos^2x = cos^2\alpha \\ tan^2x = tan^2\alpha \end{cases}$
