1. Union Properties
	-  A U B = B U A
	- A U A = A
	- A U Φ = A 
	- A U U = U
	
2. Intersection Properties
	- A ∩ B = B ∩ A
	- A ∩ A = A
	- A ∩ Φ = Φ 
	- A ∩ U = U
	
4. Assosiative Property
	- (A U B) C = A U (B U C)
	- (A ∩ B) C = A ∩ (B ∩ C)
	
5. Assosiative Property
	- A U (B ∩ C)= (A U B) ∩ (A U C)
	- A ∩ (B U C)= (A ∩ B) U (A ∩ C)
	
6. De-Morgan's Law
	- (A U B)' = A' ∩ B'
	- (A ∩ B)' = A' U B'
	
7. Compliment Properties
	- ( A')' = A
	- A ∩ A' = Φ
	- A U A' = U
